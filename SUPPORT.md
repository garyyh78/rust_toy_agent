## Getting Help

### Documentation

- [README.md](./README.md) - Quickstart and overview

### Issue Tracker

Use GitHub Issues for:
- Bug reports
- Feature requests
- Documentation improvements

**Before filing an issue:**
- Search existing issues to avoid duplicates
- Provide reproduction steps if it's a bug
- Include your environment (OS, Rust version, etc.)

### Discu

This project does not have GitHub Discussions enabled. Use Issues for questions.

## Out of Scope

- General programming questions not specific to this project
- Debugging your own code or agents
- Feature requests for specific use cases (open issues instead)

## Commercial Support

This is an open-source project maintained in spare time. There's no commercial support guarantee.
